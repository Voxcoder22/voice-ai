# Python Chat Application

This is a simple chat application built with Python using sockets and threading.

## Features

- Multiple clients can connect to the server
- Clients can send and receive messages in real-time
- Server broadcasts messages to all connected clients
- Usernames are displayed with each message

## How to Use

1. Start the server by running `python chat_server.py`
2. In separate terminals, start multiple clients by running `python chat_app.py`
3. Enter your username when prompted
4. Start chatting!

## Requirements

- Python 3.x

## Files

- `chat_server.py`: The server implementation
- `chat_app.py`: The client implementation
